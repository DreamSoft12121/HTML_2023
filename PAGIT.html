Comandos Git
init status branch checkout remote pull push clone commit

init

El comando git init se utiliza para inicializar un nuevo repositorio Git en el directorio actual.

Ejemplo:

			$ git init

status

El comando `git status` muestra el estado actual del repositorio, incluyendo cualquier cambio que no haya sido agregado o confirmado.

Ejemplo:

			$ git status

branch

El comando git branch permite crear, listar o eliminar ramas en el repositorio Git.

Ejemplo:

			$ git branch nombre_de_la_rama

checkout

El comando git checkout se utiliza para cambiar a una rama espec√≠fica o crear una nueva rama y cambiar a ella.

Ejemplo:

      		$ git checkout nombre_de_la_rama

remote

El comando git remote permite administrar los repositorios remotos asociados con un repositorio local.

Ejemplo:

      		$ git remote add nombre_del_remoto url_del_repositorio

pull

El comando git pull se utiliza para descargar cambios de un repositorio remoto y fusionarlos con la rama actual.

Ejemplo:

     		$ git pull nombre_del_remoto nombre_de_la_rama

push

El comando git push se utiliza para enviar cambios locales a un repositorio remoto.

Ejemplo:

     		$ git push nombre_del_remoto nombre_de_la_rama

push

El comando git clone se utiliza para clonar un repositorio remoto en un directorio local.

Ejemplo:

   		$ git clone url_del_repositorio

clone

El comando git clone se utiliza para clonar un repositorio remoto en un directorio local.

Ejemplo:

   		$ git clone url_del_repositorio

commit

El comando git commit se utiliza para guardar los cambios realizados en el repositorio local.

Ejemplo:

  		$ git commit -m "mensaje del commit"
